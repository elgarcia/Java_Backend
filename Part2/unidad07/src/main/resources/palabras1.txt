'''
Ejercicio 1:
Hay que crear un programa que solicite los datos de un cliente:
   Su nombre.
   Su día, mes y año de nacimiento.
Hay que almacenar los datos en variables.
A continuación debemos mostrarle un mensaje personalizado indicando algo como:

      Saludos Juan, ya sé que tienes 20 años de edad

Crea el programa usando funciones. Una función se encargará de leer los datos, otra de calcular la edad del cliente y otra de mostrar el mensaje personalizado.

Ten en cuenta que puedes obtener al año, mes y día actuales con las siguientes instrucciones:

from datetime import date
hoy = date.today()
añoActual = hoy.year
mesActual = hoy.month
diaActual = hoy.day
'''


def solicitarDatos():
    nombre = input("Tu nombre: ")
    dia = int( input("Día de nacimiento: ") )
    mes = int( input("Mes de nacimiento: ") )
    año = int( input("Año de nacimiento: ") )
    return nombre, dia, mes, año
    #return "Juan", 18, 10, 2000  


def calcularEdad(day, month, year):
    from datetime import date
    hoy = date.today()
    añoActual = hoy.year
    mesActual = hoy.month
    diaActual = hoy.day
    
    edad =  añoActual-year
    if (mesActual < month)  or (mesActual == month and diaActual < day) :
        edad = edad - 1
    return edad


def mostrarMensaje(name, edad):
    print(f"Saludos {name}, ya sé que tienes {edad} de edad")


name, day, month, year = solicitarDatos()
edad = calcularEdad(day, month, year)
mostrarMensaje(name, edad)
